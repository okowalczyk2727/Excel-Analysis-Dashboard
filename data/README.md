# Dane źródłowe

Plik [`Dane_surowe_projekt_Excel.xlsx`](Dane_surowe_projekt_Excel.xlsx) zawiera syntetyczne dane wejściowe wykorzystane w projekcie.

Dane obejmują sprzedaż, produkty, klientów, zwroty i reklamacje, budżet oraz stany magazynowe. Zostały celowo przygotowane z problemami jakościowymi, aby umożliwić praktyczne wykorzystanie Power Query i kontroli danych.

Plik należy traktować jako źródło. Transformacje powinny być wykonywane w skoroszycie z katalogu `project`, a nie bezpośrednio w danych surowych.

Skoroszyt analityczny odświeża dane przez Power Query. Model finansowo-controllingowy zawiera uzgodnione agregaty i wyłączenia kontrolne potrzebne do odtworzenia wykonania 2025 oraz planu 2026.
