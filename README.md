# Excel Sales Analysis Dashboard

## Opis projektu

Projekt przedstawia kompletny proces analizy danych sprzedażowych w programie Microsoft Excel — od przygotowania i oczyszczenia surowych danych, przez budowę modelu danych, aż do utworzenia tabel przestawnych i interaktywnego dashboardu.

Dane są syntetyczne, ale zostały przygotowane tak, aby przypominały rzeczywiste eksporty z systemów sprzedażowych, CRM i magazynowych. Zawierają celowe problemy jakościowe wymagające wykorzystania Power Query oraz funkcji Excela.

## Cel projektu

Celem projektu jest przeprowadzenie analizy sprzedaży za 2025 rok i odpowiedź między innymi na następujące pytania:

- jak zmieniała się sprzedaż i marża w kolejnych miesiącach,
- które kategorie i produkty osiągały najlepsze wyniki,
- które kanały sprzedaży były najbardziej rentowne,
- jak rzeczywiste wyniki wypadały na tle budżetu,
- jaki był poziom zwrotów i reklamacji,
- które produkty wymagają uzupełnienia zapasów,
- którzy handlowcy i klienci generowali największą wartość sprzedaży.

## Wykorzystane narzędzia

## Zbiory danych

Plik źródłowy zawiera dane dotyczące:

- sprzedaży detalicznej i B2B,
- klientów,
- produktów i cen,
- zwrotów i reklamacji,
- budżetu sprzedażowego,
- stanów magazynowych.

Dane zawierają między innymi duplikaty, puste rekordy, niespójne nazwy, różne formaty dat, kwot i procentów oraz błędnie zapisane kody produktów.
