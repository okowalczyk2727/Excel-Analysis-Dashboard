# Skoroszyty projektu

## Analiza_Sprzedazy_Excel.xlsx

Skoroszyt analityczny zawiera:

- zapytania Power Query,
- tabele mapujące i kontrolne,
- model danych Power Pivot,
- miary DAX,
- tabele i wykresy przestawne,
- fragmentatory kategorii i kanału,
- oś czasu,
- arkusz `DASHBOARD`,
- analizy czasu, kategorii, kanałów, zwrotów i magazynu.

Plik: [`Analiza_Sprzedazy_Excel.xlsx`](Analiza_Sprzedazy_Excel.xlsx)

## Model_Finansowo_Controllingowy.xlsx

Model finansowy rozwija wykonanie 2025 do planu na 2026 rok. Zawiera:

- uzgodnienie danych źródłowych,
- jawne wyłączenia rekordów `KONTROLA_*`,
- założenia dla trzech scenariuszy,
- budżet 2025 i plan miesięczny 2026,
- rachunek wyników,
- analizę odchyleń,
- analizę ceny, wolumenu i miksu,
- zatrudnienie, OPEX, CAPEX i amortyzację,
- kapitał obrotowy i przepływy pieniężne,
- próg rentowności,
- panel zarządczy,
- 20 kontroli spójności.

Plik: [`Model_Finansowo_Controllingowy.xlsx`](Model_Finansowo_Controllingowy.xlsx)

## Wymagania

Do pełnej obsługi projektu potrzebny jest Microsoft Excel Desktop dla Windows z dodatkiem Power Pivot. Excel Online nie zapewnia pełnej obsługi modelu danych, miar DAX, fragmentatorów i osi czasu.

Po przeniesieniu repozytorium Power Query może poprosić o wskazanie nowej ścieżki do pliku `../data/Dane_surowe_projekt_Excel.xlsx`.
