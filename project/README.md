# Skoroszyt analityczny

Główny plik projektu:

[`Analiza_Sprzedazy_Excel.xlsx`](Analiza_Sprzedazy_Excel.xlsx)

Skoroszyt zawiera:

- zapytania Power Query,
- tabele mapujące i kontrolne,
- model danych Power Pivot,
- miary DAX,
- tabele i wykresy przestawne,
- arkusz `DASHBOARD` z fragmentatorami i osią czasu.

Do pełnej obsługi projektu wymagany jest **Microsoft Excel Desktop dla Windows** z dodatkiem Power Pivot. Excel Online może wyświetlić część zawartości, ale nie zapewnia pełnej obsługi modelu danych i miar DAX.

Po przeniesieniu repozytorium Power Query może wymagać wskazania nowej ścieżki do pliku `../data/Dane_surowe_projekt_Excel.xlsx`.
